# Code of Conduct

Lumen Browser aims to be a practical, respectful open-source project.

## Expected Behavior

- Be direct and constructive.
- Focus criticism on code, behavior, documentation, and tradeoffs.
- Assume maintainers and contributors have limited time.
- Respect privacy and security disclosure boundaries.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Public disclosure of private security reports before maintainers have had time to respond.
- Posting private data, credentials, tokens, browsing data, or personal contact details without consent.
- Spam or off-topic promotion.

## Enforcement

Maintainers may edit, hide, lock, or remove comments, issues, pull requests, or participation that violates this code. Serious or repeated violations may result in a ban from project spaces.

Report conduct concerns privately through the repository owner profile.
